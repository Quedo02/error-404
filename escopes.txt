Proceso general

    Configuración de Rutas:
        En el index.php, establecer las rutas para cada vista utilizando el patrón MVC.
        Asegurarse de que las rutas apunten a los controladores correctos.

    Desarrollo de la Vista:
        Crear las vistas HTML/CSS/JS para cada uno de los roles de usuario.
        Utilizar AJAX o fetch para conectar las vistas con las APIs del backend.

    Aprendizaje en equipo:
        Realizar sesiones de pair programming para que todos aprendan tanto el backend como el frontend.
        Compartir los conocimientos adquiridos y hacer revisiones de código entre los integrantes del equipo.
